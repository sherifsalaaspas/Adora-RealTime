# 📦 Adora‑RealTime
### Unified Shipping Intelligence — Open, Modern, Real‑Time

Adora‑RealTime is a **non‑profit, open‑source shipping aggregator** that replicates the workflow of *MyDHL Express* while adding:

- HS Code Autocomplete  
- Multi-carrier orchestrated rate fetching  
- Intelligent “Best Value” recommendation engine  
- Unified tracking experience  
- Real carrier API support (optional using env keys)  

Designed for instant deployment on **Vercel**.

---

## 🎨 Tech Stack

- **React 18**  
- **Vite**  
- **TailwindCSS**  
- **Vercel Functions** (Node handlers)  
- **Demo Mode** with realistic simulated carrier pricing
